## Installation

The package can be installed by adding `membrane_rtc_engine_rtsp` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:membrane_rtc_engine_rtsp, "~> 0.1.0"}
  ]
end
```
